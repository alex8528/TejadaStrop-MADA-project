# Analysis

This folder contains scripts used for the main statistical analysis and model validation for the project.

## Files

- `04_statistical_analysis.qmd`  
  Fits the main bivariate and multivariable logistic regression models and saves model outputs.

- `05_model_validation.qmd`  
  Evaluates model performance using 5 fold cross validation and compares logistic regression with random forest.

## Notes

These scripts should be run after the data processing and exploratory analysis scripts.

Outputs generated by these scripts are saved in:

- `results/tables/`
- `results/output/`
