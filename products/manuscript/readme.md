# Manuscript

This folder contains the main manuscript for the MADA course project.

The manuscript is written using **Quarto** and is designed to be fully reproducible.  
All figures, tables, and results are generated from analysis scripts and stored outputs located elsewhere in the repository.

The manuscript focuses on the association between **prenatal mental health screening and postpartum depressive symptoms** using PRAMS Phase 8 data.
