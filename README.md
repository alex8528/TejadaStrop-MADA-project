# Prenatal Mental Health Screening and Postpartum Depressive Symptoms

## Overview
This project looks at whether prenatal mental health screening is associated 
with postpartum depressive symptoms, using data from the Pregnancy Risk 
Assessment Monitoring System (PRAMS) Phase 8.

The main question is: Are individuals who were asked about depression or 
anxiety during prenatal care less likely to report postpartum depressive 
symptoms?

## How the project is organized
The analysis runs in a step-by-step pipeline:

raw data → cleaning → analytic dataset → exploratory analysis → modeling → manuscript

All outputs (tables, figures, model results) are generated by code and saved 
to the results folder.

## How to reproduce the analysis

Run the scripts in this order. Each one needs to finish before starting the next.

1. `code/processing/01_load_clean.qmd`
   - Loads and cleans the raw PRAMS data
   - Output: `data/processed-data/prams_clean.rds`
   - Note: If the raw data file is not available, this script will skip 
     processing. The file `prams_analytic_localfix.rds` is already included 
     in the repo and is what everything else uses.

2. `code/processing/02_create_analytic_dataset.qmd`
   - Creates the analytic dataset used for modeling
   - Output: `data/processed-data/prams_analytic_localfix.rds`

3. `code/eda/03_exploratory_analysis.qmd`
   - Descriptive tables and figures
   - Output: `results/figures/screening_vs_postpartum_depression.png`
   - Output: `results/tables/screening_distribution.rds`

4. `code/analysis/04_statistical_analysis.qmd`
   - Bivariate and multivariable logistic regression
   - Output: `results/tables/glm_bivariate_screened.rds`
   - Output: `results/tables/glm_multivariable_main.rds`

5. `code/analysis/05_model_validation.qmd`
   - 5-fold cross-validation comparing logistic regression and random forest
   - Output: `results/tables/model_comparison.rds`
   - Output: `results/figures/model_performance.png`

6. `quarto render products/manuscript/Manuscript.qmd`
   - Renders the final manuscript using all saved outputs

## Data

Raw data live in `data/raw-data/`. All processed files are in 
`data/processed-data/` and saved as `.rds` files to preserve variable types.
